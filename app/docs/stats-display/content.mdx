Agents often surface numerical summaries: revenue this quarter, active users, error rates. StatsDisplay renders those numbers in a responsive grid with sparklines, delta indicators, and locale-aware formatting. Without it, the assistant would list the figures in prose.

## Getting Started

<Steps>

<Step title="Install the component">

Run this once from your project root.

```bash
npx shadcn@latest add @tool-ui/stats-display
```

</Step>

<Step title="Use it in your application">

Render `StatsDisplay` in your UI with tool-compatible props.

<div className="[&_pre]:max-h-[30rem] [&_pre]:overflow-y-auto">

```tsx
import { StatsDisplay } from "@/components/tool-ui/stats-display";

export function Example() {

      description="October through December 2024"

        {

          sparkline: { data: [18200, 19100, 19800, 20400, 21200, 21900, 22600, 23100, 23800, 24200, 24500, 24890], color: "var(--chart-3)" },
          diff: { value: 8.2, decimals: 1 },
```

<Step title="Register it in your runtime provider">

Register this renderer so tool results display as `StatsDisplay`.

```tsx
"use client";

import { type Toolkit } from "@assistant-ui/react";
import { StatsDisplay } from "@/components/tool-ui/stats-display";
import {
  safeParseSerializableStatsDisplay,
  SerializableStatsDisplaySchema,
} from "@/components/tool-ui/stats-display/schema";
import { createArgsToolRenderer } from "@/components/tool-ui/shared";

export const toolkit: Toolkit = {
  showStats: {
    description: "Display key metrics in a grid",
    parameters: SerializableStatsDisplaySchema,
    render: createArgsToolRenderer({
      safeParse: safeParseSerializableStatsDisplay,
      idPrefix: "stats",
      render: (parsedArgs) => (
          <StatsDisplay {...parsedArgs} />
      ),
    }),
  },
};
```

</Step>

</Steps>

## Key Features

<FeatureGrid>
  <Feature icon="Grid2x2" title="Auto-responsive grid">
    Tiles stats in columns that reflow to fit the container
  </Feature>
  <Feature icon="LineChart" title="Inline sparklines">
    Pure SVG trend lines with no extra dependencies
  </Feature>
  <Feature icon="BarChart3" title="Delta indicators">
    Green for up, red for down (flip with `upIsPositive: false` for metrics like churn)
  </Feature>
  <Feature icon="DollarSign" title="Format options">
    Currency, percent, number, and text with locale-aware formatting
  </Feature>
  <Feature icon="Target" title="Single-stat hero mode">
    Pass one stat for a larger, centered card
  </Feature>
</FeatureGrid>
