When the assistant generates or references code, unstyled monospace text sells it short. CodeBlock renders syntax-highlighted snippets with line numbers and a copy button. You can highlight specific lines to draw attention to changes or bugs, and collapse long snippets with `maxCollapsedLines` to keep conversations scannable.

**Role:** Information — displays data for the user to read. See [Design Guidelines](/docs/design-guidelines) for how component roles work.

## Getting Started

<Steps>

<Step title="Install the component">

Run this once from your project root.

```bash
npx shadcn@latest add @tool-ui/code-block
```

</Step>

<Step title="Use it in your application">

Render `CodeBlock` in your UI with tool-compatible props.

<div className="[&_pre]:max-h-[30rem] [&_pre]:overflow-y-auto">

```tsx
import { CodeBlock } from "@/components/tool-ui/code-block";

export function MyComponent() {
  return (
    <CodeBlock.Standard
      id="my-code-example"
      code={`function hello(name: string) {
  return \`Hello, \${name}!\`;
}`}
      language="typescript"
      lineNumbers="visible"
      filename="hello.ts"
    />
  );
}
```

</div>

</Step>

<Step title="Register it in your runtime provider">

Register this renderer so tool results display as `CodeBlock`.

```tsx
// Backend tool
import { tool, jsonSchema } from "ai";

const showCodeBlock = tool({
  description: "Show a code snippet",
  inputSchema: jsonSchema<{}>({
    type: "object",
    properties: {},
    additionalProperties: false,
  }),
  async execute() {
    return {
      id: "code-block-1",
      code: "console.log('hello');",
      language: "javascript",
      filename: "hello.js",
      highlightLines: [1],
    };
  },
});

// Frontend with assistant-ui
import { type Toolkit } from "@assistant-ui/react";
import { CodeBlock } from "@/components/tool-ui/code-block";
import { safeParseSerializableCodeBlock } from "@/components/tool-ui/code-block/schema";
import { createResultToolRenderer } from "@/components/tool-ui/shared";

export const toolkit: Toolkit = {
  showCodeBlock: {
    type: "backend",
    render: createResultToolRenderer({
      safeParse: safeParseSerializableCodeBlock,
      render: (parsedResult) => (
        <CodeBlock.Standard {...parsedResult} />
      ),
    }),
  },
};
```

</Step>

</Steps>

## Supported Languages

CodeBlock supports all languages included with Shiki, including:

- TypeScript/JavaScript
- Python
- JSON
- Bash/Shell
- CSS/HTML
- Markdown
- SQL
- YAML
- Go
- Rust
- And many more...

See [Shiki documentation](https://shiki.style/languages) for the complete list.

## Accessibility

- Collapsible sections use Radix Collapsible with proper ARIA attributes
- Copy button is keyboard-accessible with focus indication
- Code content is selectable and works with screen readers
