## Getting Started

<Steps>

<Step title="Install the component">

Run this once from your project root.

```bash
npx shadcn@latest add https://tool-ui.com/r/instagram-post.json
```

</Step>

<Step title="Use it in your application">

Render `InstagramPost` in your UI with tool-compatible props.

<div className="[&_pre]:max-h-[30rem] [&_pre]:overflow-y-auto">

```tsx
import { InstagramPost } from "@/components/tool-ui/instagram-post";

export function Example() {
  return (
    <InstagramPost
      post={{
        id: "ig-post-1",
        author: {
          name: "Alex Rivera",
          handle: "alexrivera",
          avatarUrl: "https://example.com/avatar.jpg",
          verified: true,
        },
        text: "Golden hour vibes in the city.",
        media: [
          {
            type: "image",
            url: "https://example.com/post-image.jpg",
            alt: "City skyline",
          },
        ],
        stats: { likes: 8421 },
        createdAt: "2025-01-05T18:00:00.000Z",
      }}
    />
  );
}
```

</div>

</Step>

<Step title="Register it in your runtime provider">

Register this renderer so tool results display as `InstagramPost`.

```tsx
import { type Toolkit } from "@assistant-ui/react";
import {
  InstagramPost,
  safeParseSerializableInstagramPost,
} from "@/components/tool-ui/instagram-post";
import { createResultToolRenderer } from "@/components/tool-ui/shared";

export const toolkit: Toolkit = {
  showInstagramPost: {
    type: "backend",
    render: createResultToolRenderer({
      safeParse: safeParseSerializableInstagramPost,
      render: (parsedResult) => <InstagramPost post={parsedResult} />,
    }),
  },
};
```

</Step>

</Steps>

These snippets use [assistant-ui](https://assistant-ui.com) for end-to-end wiring, but the InstagramPost component itself is framework-agnostic at the UI layer: you can use it in any React codebase with any LLM SDK or tool-calling interface that can provide compatible props.
