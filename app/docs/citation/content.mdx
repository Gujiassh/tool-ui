Citation renders a clickable reference card with title, domain, favicon, and an optional text snippet. Use it when the assistant cites a source and you want the user to see where the information came from. Three variants cover different density needs: a full card, an inline chip, or stacked favicons.

## Getting Started

<Steps>

<Step title="Install the component">

Run this once from your project root.

```bash
npx shadcn@latest add @tool-ui/citation
```

</Step>

<Step title="Use it in your application">

Render `Citation` in your UI with tool-compatible props.

<div className="[&_pre]:max-h-[30rem] [&_pre]:overflow-y-auto">

```tsx
import { Citation } from "@/components/tool-ui/citation";

export function Example() {
  return (
    <Citation
      id="citation-example"
      href="https://react.dev/reference/react/useState"
      title="useState – React"
      snippet="useState is a React Hook that lets you add a state variable to your component."
      domain="react.dev"
      favicon="https://www.google.com/s2/favicons?domain=react.dev&sz=32"
      type="document"
    />
  );
}
```

</div>

</Step>

<Step title="Register it in your runtime provider">

Register this renderer so tool results display as `Citation`.

```tsx
import { type Toolkit } from "@assistant-ui/react";
import { Citation } from "@/components/tool-ui/citation";
import { createResultToolRenderer } from "@/components/tool-ui/shared";

export const toolkit: Toolkit = {
  showCitation: {
    type: "backend",
    render: createResultToolRenderer({
      render: (parsedResult) => <Citation {...parsedResult} />,
    }),
  },
};
```

</Step>

</Steps>

## Accessibility

The stacked variant popover supports full keyboard navigation:

- **Enter / Space:** Open the popover
- **Tab:** Navigate between citations
- **Escape:** Close popover and return focus to trigger

Mouse users can hover to reveal the popover. All variants open links in new tabs with `noopener,noreferrer` for security.
